== 物件成員

此物件包含下列成員：

|===
| 名稱 | 說明 | 類型 


 a| 
AggrName
 a| 
指派給Vserver的Aggregate名稱。
 a| 
字串



 a| 
AggrAvailable大小
 a| 
指派的Aggregate可用大小。
 a| 
整數

|===
