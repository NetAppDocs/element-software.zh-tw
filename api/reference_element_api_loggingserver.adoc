= 記錄伺服器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
loggingServer 物件儲存有關為儲存叢集配置的日誌主機的資訊。使用 `GetRemoteLoggingHosts`查看目前日誌記錄主機，然後使用 `SetRemoteLoggingHosts`設定目前和新增日誌主機的所需清單。



== 物件成員

此物件包含下列成員：

|===
| 名稱 | 說明 | 類型 


 a| 
主機
 a| 
日誌伺服器的 FQDN 或 IP 位址
 a| 
字串



 a| 
連接埠
 a| 
用於與日誌伺服器通訊的連接埠號
 a| 
整數

|===
