= SetSSLCertificate
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「etSSLCertificate」方法、為叢集中的儲存節點設定使用者SSL憑證和私密金鑰。


NOTE: 使用API之後、您必須重新開機管理節點。



== 參數

此方法具有下列輸入參數：

|===
| 名稱 | 說明 | 類型 | 預設值 | 必要 


 a| 
憑證
 a| 
憑證的PEEM編碼文字版本。*附註：*設定節點或叢集憑證時、憑證必須包含serverauth的extendedKeyusage副檔名。此擴充功能可讓您在一般作業系統和瀏覽器上使用憑證、而不會發生錯誤。如果沒有副檔名、API將拒絕該憑證為無效。
 a| 
字串
 a| 
無
 a| 
是的



 a| 
私有金鑰
 a| 
私密金鑰的PEEM編碼文字版本。
 a| 
字串
 a| 
無
 a| 
是的

|===


== 傳回值

此方法沒有傳回值。
