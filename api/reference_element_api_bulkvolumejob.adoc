= 艙壁Volume工作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
BulkVolume工作物件包含大量磁碟區讀取或寫入作業的相關資訊、例如複製或建立快照。



== 物件成員

此物件包含下列成員：

 a|

 a| 
整數



 a| 
CREATETIM
 a| 
為大量Volume工作建立的時間戳記、格式為UTC+0。
 a| 
ISO 8601日期字串



 a| 
所經過的時間
 a| 
自工作開始以來的秒數。
 a| 
字串



 a| 
格式
 a| 
大量Volume作業的格式。可能值：

* 壓縮
* 原生

字串



 a|

 a|

字串



 a|
