= 叢集資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
clusterInfo物件包含節點用來與叢集通訊的資訊。您可以使用GetClusterInfo API方法取得此資訊。



== 物件成員

此物件包含下列成員：

| 名稱 | 說明 | 類型 


 a| 
屬性
 a|

 a| 
預設保護方案
 a|

 a| 
字串



 a| 
enabledProtectionSchemes
 a| 
已在此儲存叢集上啟用的所有保護配置清單。

* 啟用：正在啟用靜止加密。

* 停用：停用靜止加密。
