= 叢集API方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element軟體叢集API方法可讓您管理儲存叢集的組態和拓撲、以及屬於儲存叢集的節點。

某些叢集API方法會在屬於叢集一部分的節點上運作、或是已設定加入叢集。您可以將節點新增至新叢集或現有叢集。準備新增至叢集的節點處於「擱置中」狀態、表示已設定這些節點、但尚未新增至叢集。
