= 停用Ssh
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「disableSsh」方法停用單一儲存節點的SSH服務。此方法不會影響整個叢集的SSH服務逾時持續時間。


NOTE: 不支援此方法。SolidFire如果SolidFire 您在EESDS系統上使用此方法、則該方法會傳回錯誤訊息。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有下列傳回值：

|===
| 名稱 | 說明 | 類型 


 a| 
已啟用
 a| 
此節點的SSH服務狀態。
 a| 
布林值

|===


== 申請範例

此方法的要求類似於下列範例：

[listing]
----
{
  "method": "DisableSsh",
  "params": {
      },
  "id" : 1
}
----


== 回應範例

此方法會傳回類似下列範例的回應：

[listing]
----
{
  "id" : 1,
  "result" : {"enabled": false}
}
----
