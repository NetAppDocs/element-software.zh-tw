= 使用Element API管理儲存設備
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用Element軟體API來管理元素儲存叢集。

元素API是以透過HTTPS的JSON-RPC傳輸協定為基礎。JSON-RPC是以輕量化JSON資料交換格式為基礎的簡單文字型RPC傳輸協定。用戶端程式庫適用於所有主要的程式設計語言。
