= API驗證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用API時、您可以在所有API要求中加入HTTP Basic驗證標頭、以驗證系統。如果您省略驗證資訊、系統會以HTTP 401回應拒絕未驗證的要求。系統支援透過TLS進行HTTP Basic驗證。

使用叢集管理帳戶進行API驗證。
