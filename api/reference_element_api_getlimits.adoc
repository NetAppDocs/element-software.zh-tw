= GetLimits
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「GetLimits」方法來取得API設定的限制值。這些值可能會在元素的版本之間變更、但若未更新系統、則不會變更。瞭解API設定的限制值、在撰寫面向使用者工具的API指令碼時非常實用。


NOTE: 無論用於傳遞方法的API端點版本為何、「GetLimits」方法都會傳回目前軟體版本的限制。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法會傳回含有API限制之名稱值配對的Json物件。



== 申請範例

此方法的要求類似於下列範例：

[listing]
----
{
   "method": "GetLimits",
   "id" : 1
}
----
