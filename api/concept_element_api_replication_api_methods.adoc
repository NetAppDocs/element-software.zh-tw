= 複寫API方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
複寫API方法可讓您連接兩個叢集、以實現持續資料保護（CDP）。當您連接兩個叢集時、叢集內的作用中磁碟區可持續複寫至第二個叢集、以提供資料還原功能。將磁碟區配對以進行複寫、即可保護資料免受可能導致資料無法存取的事件影響。
