= GetSSLCertificate
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「GetSSLCertificate」方法擷取叢集儲存節點上目前使用中的SSL憑證。



== 參數

此方法沒有輸入參數。



== 傳回值

此方法具有下列傳回值：

|===
| 名稱 | 說明 | 類型 


 a| 
憑證
 a| 
憑證的完整PEE編碼文字。
 a| 
字串



 a| 
詳細資料
 a| 
憑證的解碼資訊。
 a| 
Json物件

|===


== 申請範例

此方法的要求類似於下列範例：

[listing]
----
{
    "method" : "GetSSLCertificate",
    "params" : {},
    "id" : 1
}
----
