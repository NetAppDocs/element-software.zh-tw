= 刪除虛擬網路
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以移除虛擬網路物件。在移除虛擬網路之前、您必須先將位址區塊新增至其他虛擬網路。

. 按一下*叢集*>*網路*。
. 按一下您要刪除之VLAN的「動作」圖示。
. 按一下*刪除*。
. 確認訊息。
