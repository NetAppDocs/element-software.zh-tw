= 使用Volume
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
這個系統使用Volume來配置儲存設備。SolidFire磁碟區是透過iSCSI或Fibre Channel用戶端透過網路存取的區塊裝置。在「管理」索引標籤的「磁碟區」頁面中、您可以在節點上建立、修改、複製及刪除磁碟區。您也可以檢視磁碟區頻寬和I/O使用量的統計資料。
