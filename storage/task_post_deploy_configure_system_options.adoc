= 部署後設定SolidFire 系統選項
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
設定SolidFire 完整套系統之後、您可能會想要執行一些選用的工作。

如果您變更系統中的認證資料、您可能會想知道對其他元件的影響。

此外、您也可以設定多因素驗證、外部金鑰管理及聯邦資訊處理標準（FIPS）安全性的設定。您也應該視需要更新密碼。
