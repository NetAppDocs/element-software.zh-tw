= SolidFire Active IQ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
支援以網路為基礎的工具、可持續更新整個叢集資料的歷史檢視。Active IQ您可以針對特定事件、臨界值或度量設定警示。利用此功能、您可以監控系統效能與容量、並隨時掌握叢集健全狀況。Active IQ

您可以在Active IQ 下列關於您系統的資訊中找到：

* 節點數量和節點狀態：健全、離線或故障
* CPU與記憶體使用量的圖形化表示
* 關於節點的詳細資料、例如序號、機箱中的插槽位置、模型、NetApp Element 以及儲存節點上執行的版本介紹
* 有關虛擬機器的CPU與儲存相關資訊
