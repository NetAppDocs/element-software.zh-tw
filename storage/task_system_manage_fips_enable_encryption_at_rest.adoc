= 在靜止狀態下啟用加密
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在閒置時啟用和停用全叢集加密。此功能預設為未啟用。若要支援FIPS磁碟機、您必須在閒置時啟用加密。

. 在這個軟件UI中、按一下NetApp Element *叢集*>*設定*。
. 按一下「*在REST啟用加密*」。
