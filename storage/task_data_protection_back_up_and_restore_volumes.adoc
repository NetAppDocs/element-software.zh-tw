= 備份與還原磁碟區
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以將磁碟區備份並還原至其他SolidFire 的不一致儲存設備、以及與Amazon S3或OpenStack Swift相容的次要物件存放區。

從OpenStack Swift或Amazon S3還原磁碟區時、您需要原始備份程序的資訊清單資訊。如果您要還原SolidFire 已備份到還原系統上的磁碟區、則不需要任何資訊清單資訊。
