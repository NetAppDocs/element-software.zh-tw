= 管理帳戶
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在支援功能的儲存系統中、使用者帳戶可讓用戶端連線至節點上的磁碟區。SolidFire當您建立Volume時、該磁碟區會指派給特定的使用者帳戶。您也可以管理SolidFire 適用於某個不實儲存系統的叢集管理員帳戶。
