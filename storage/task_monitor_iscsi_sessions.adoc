= 檢視iSCSI工作階段
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢視連線至叢集的iSCSI工作階段。您可以篩選資訊、僅包含所需的工作階段。

. 在Element UI中、選取* Reports*>* iSCSI工作階段*。
. 若要查看篩選條件欄位、請按一下*篩選*。
