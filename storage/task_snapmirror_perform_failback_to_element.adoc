= 執行容錯回復至元素
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當主要方面的問題已經解決時、您必須重新同步原始來源Volume並容錯回復NetApp Element 至該軟體。您執行的步驟會因原始來源磁碟區是否仍存在或是否需要容錯回復至新建立的磁碟區而有所不同。
