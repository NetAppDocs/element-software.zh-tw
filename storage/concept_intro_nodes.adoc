= 節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
節點是指群組為叢集的個別硬體元件、可作為區塊儲存設備進行存取。在一個不二的儲存系統中、有兩種類型的節點SolidFire ：儲存節點和光纖通道節點。
