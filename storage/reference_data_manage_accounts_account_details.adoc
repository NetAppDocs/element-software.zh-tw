= 帳戶詳細資料
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「管理」索引標籤上的「帳戶」頁面提供系統中每個帳戶的相關資訊、例如指派給每個帳戶之磁碟區的ID、使用者名稱及效率詳細資料。

* *識別碼*
+
系統產生的帳戶ID。

* *使用者名稱*
+
建立帳戶時所指定的名稱。

* *狀態*
+
帳戶狀態。可能值：

+
** 作用中：作用中帳戶。
** 鎖定：鎖定的帳戶。
** 已移除：已刪除及清除的帳戶。


* *作用中磁碟區*
+
指派給帳戶的作用中磁碟區數目。

* *壓縮*
+
指派給帳戶之磁碟區的壓縮效率分數。

* *重複資料刪除*
+
指派給帳戶之磁碟區的重複資料刪除效率分數。

* *精簡配置*
+
指派給帳戶之磁碟區的精簡配置效率分數。

* *整體效率*
+
指派給帳戶之磁碟區的整體效率分數。
