= 管理帳戶
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在功能性儲存系統中、租戶可以使用帳戶來讓用戶端連線至叢集上的磁碟區。SolidFire當您建立Volume時、該磁碟區會指派給特定帳戶。您也可以管理SolidFire 適用於某個不實儲存系統的叢集管理員帳戶。
