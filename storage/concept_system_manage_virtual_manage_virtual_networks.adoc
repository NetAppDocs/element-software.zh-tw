= 管理虛擬網路
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
利用支援虛擬網路SolidFire 功能的虛擬網路功能、可將位於不同邏輯網路上的多個用戶端之間的流量連線至一個叢集。透過使用VLAN標記、將與叢集的連線隔離在網路堆疊中。
