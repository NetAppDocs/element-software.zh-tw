= 設定叢集設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從元素UI的「叢集」索引標籤檢視及變更整個叢集的設定、並執行叢集特定的工作。

您可以設定叢集完整度臨界值、支援存取、閒置加密、虛擬磁碟區、SnapMirror、 和NTP廣播用戶端。
